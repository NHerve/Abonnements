Command avec CTRL ... + CTR
	Collasping / Expanding
		M + O = COLLASPE ALL
		M + L = EXPAND ALL
		M + M = Collaspe / Expand Line
	Commenting : 
		K + C = Comment
		K + U = Uncomment
	Moving Command / Block:
		ALT UP : Moving UP
		ALT DOWN : MOVING DOWN
